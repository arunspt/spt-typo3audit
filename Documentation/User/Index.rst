Users Manual
============

1. Install the extension using extension manager.
2. Include the extension in the root page static template.
3. You can see a new backend module named “Audit Manager”.
4. Click on the backend module to see the details related to the server and the extensions installed in the TYPO3 system.
5. By clicking on the PDF icon, it is possible to export the report as PDF
